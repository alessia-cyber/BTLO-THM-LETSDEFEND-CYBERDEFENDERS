# TryHackMe- Cloud Security Pitfalls 


## Overview

This report details my analysis and findings from the Cloud Security Pitfalls room on TryHackMe.
The room covers the risks companies face when migrating to the cloud and outlines strategies 
to identify and address them within a Security Operations Center (SOC).


## Skills Acquired

- **Understood Core Cloud Models**: Differentiated between IaaS, PaaS, and SaaS and their specific security implications;
- **Analyzed Provider Risks**: Explored security risks that originate from cloud service providers;
- **Mastered Cloud Security Concepts**: Grasped the fundamental principles of securing cloud environments.
- **Optimized SOC Monitoring**: Identified and overcame the unique challenges of monitoring cloud infrastructure as a SOC analyst.
- **Mitigated Migration Risks**: Applied strategies to prevent common pitfalls during the transition from on-premises to the cloud.
